# 线材真实长度

## 长度分布

| 概率  | 长度范围        |
|-----|-------------|
| 50% | 0.2m ~ 0.5m |
| 35% | 0.5m ~ 1.0m |
| 15% | 1.0m ~ 2.0m |

---

# 最大长度限制

| 参数                | 默认值 |
|-------------------|-----|
| MAX_WIRE_LENGTH_M | 2.0 |

说明：

生成线材长度不会超过2m。

---

# 图像长度限制

| 参数             | 默认值  |
|----------------|------|
| MAX_WIRE_RATIO | 0.35 |

说明：

线材最长边不超过图像宽度35%。

例如：

1280宽图像：

最大长度：

1280 × 0.35

≈ 448 pixel

---

# 透视缩放

## Perspective Scale

| 参数                    | 默认值  |
|-----------------------|------|
| MIN_PERSPECTIVE_SCALE | 0.08 |
| MAX_PERSPECTIVE_SCALE | 1.20 |

说明：

远处目标更小，
近处目标更大。

---

# 远近采样策略

| 参数                  | 默认值 |
|---------------------|-----|
| DISTANCE_BIAS_POWER | 2   |

说明：

1 = 均匀采样

2 = 略偏远处

3 = 强偏远处

4 = 极强偏远处

推荐：

线材检测：

DISTANCE_BIAS_POWER = 3

---

# 旋转

| 参数         | 默认值 |
|------------|-----|
| ROTATE_MIN | 0   |
| ROTATE_MAX | 360 |

说明：

随机旋转角度。

---

# 阴影

| 参数                 | 默认值  |
|--------------------|------|
| SHADOW_STRENGTH    | 0.08 |
| SHADOW_BLUR_KERNEL | 31   |

说明：

模拟线材遮挡产生的阴影。

---

# 模糊

| 参数                | 默认值     |
|-------------------|---------|
| WIRE_BLUR_KERNELS | [0,3,5] |

说明：

模拟远距离失焦。

0 = 不模糊

3 = 轻微模糊

5 = 明显模糊

---

# 光照变化

| 参数             | 默认值 |
|----------------|-----|
| BRIGHTNESS_MIN | 0.8 |
| BRIGHTNESS_MAX | 1.2 |

说明：

模拟不同环境亮度。

---

# 重叠

| 参数                | 默认值  |
|-------------------|------|
| MAX_OVERLAP_RATIO | 0.05 |

说明：

允许5%目标重叠。

---

# 放置尝试次数

| 参数              | 默认值 |
|-----------------|-----|
| MAX_PLACE_RETRY | 100 |

说明：

超过100次无法放置则放弃。

# 推荐配置

```python

WIRE_BLUR_KERNELS = [0, 3, 5, 7]
```
